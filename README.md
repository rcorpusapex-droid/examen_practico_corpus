# Java Ejercicios + PokeAPI Pro:

Proyecto en Java realizado como prueba técnica. Incluye ejercicios prácticos, consumo de PokeAPI y una versión con Spring Boot para pruebas en local.

## Contenido

Este proyecto incluye:

- ejercicios básicos de Java
- consumo de PokeAPI desde consola
- API local con Spring Boot, se agrega contrato.
